# Personal Portfolio Website

A modern, fully responsive personal portfolio website built with React, Vite, and custom CSS. It features a mid-century modern aesthetic with bold geometric designs and smooth scrolling navigation.

## Features
- **Responsive Design**: Adapts beautifully to desktop, tablet, and mobile screens.
- **Custom Aesthetic**: Mid-century modern style with a strict geometric, retro color palette.
- **Dynamic CSS Grid**: Advanced CSS grid layouts for projecting skills and projects.
- **Single Page Application**: Fast loading and smooth scrolling navigation using React.

## Technologies Used
- React.js
- Vite
- Custom CSS Variables and Flexbox/Grid
- HTML5

## Running Locally

To run this project locally on your machine:

1. Clone the repository.
2. Navigate into the `Portfolio-Website/portfolio-site` directory.
3. Install dependencies by running `npm install`.
4. Run the development server with `npm run dev`.
5. Open your browser to the local URL provided by Vite (e.g., `http://localhost:5173/`).
